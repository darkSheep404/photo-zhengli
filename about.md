# 关于照片整理

## 项目简介

**照片整理** (Photo Zhengli) 是一款专为安卓用户设计的照片清理应用。它通过简洁直观的操作界面，帮助用户快速浏览并整理手机中的照片——保留重要的，删除不需要的，归类到合适的相册中。

## 设计理念

灵感来源于百度一刻的照片整理交互方式：

- **一张一张看**：以大图形式逐张展示照片，避免遗漏
- **三键决策**：每张照片只需做一个决定——保留、删除、或移到相册
- **安全删除**：删除的照片进入系统「最近删除」，30 天内可恢复，不怕误删
- **批量确认**：整理完毕后统一确认，避免误操作

## 核心功能

### 1. 首页概览
显示手机照片总数、相册数量和占用空间，让你了解照片存储现状。点击「开始整理」进入整理流程。

### 2. 照片整理
- 大图预览当前照片
- 底部缩略图条快速跳转
- 月份筛选器按时间段筛选
- 进度条显示当前整理进度

### 3. 三键操作
- **↩ 撤销**：撤回上一步操作
- **📁 添加到**：弹出相册列表，选择目标相册移动照片
- **🗑 删除**：标记照片为待删除

### 4. 相册管理
- 浏览现有相册列表
- 快速新建相册
- 将照片移动到指定相册

### 5. 审核确认
- 查看所有待删除和待移动的照片
- 一键确认执行
- 显示释放的存储空间

## 技术实现

### 前端
使用 Vue 3 + TypeScript 构建响应式 UI，所有界面使用 HTML/CSS/JS 实现，通过 Capacitor 打包为 Android APK。

### 原生插件
两个自定义 Capacitor 插件（Java）：

- **MovePhotoPlugin**：通过 MediaStore API 修改照片的 `RELATIVE_PATH`，实现跨相册移动
- **TrashPhotoPlugin**：
  - Android 11+：使用 `MediaStore.createTrashRequest()` 将照片移入系统「最近删除」
  - Android 10：使用 `ContentResolver.delete()` 删除（系统相册自行管理回收站）

### CI/CD
GitHub Actions 自动化流程：
- 每次推送自动构建 debug APK
- 推送 `v*` tag 自动构建 release APK 并创建 GitHub Release
